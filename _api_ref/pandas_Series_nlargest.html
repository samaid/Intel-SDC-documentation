pandas.Series.nlargest¶

Return the largest n elements.

Parameters¶

nint, default 5

Return this many descending sorted values.

keep{‘first’, ‘last’, ‘all’}, default ‘first’

When there are duplicate values that cannot all fit in a Series of n elements:

  • firstreturn the first n occurrences in order

    of appearance.

  • lastreturn the last n occurrences in reverse

    order of appearance.

  • allkeep all occurrences. This can result in a Series of

    size larger than n.

Returns¶

Series

The n largest values in the Series, sorted in decreasing order.

Examples¶

pandas.Series.nlargest example¶
import numpy as np
import pandas as pd
import sdc


@sdc.jit
def series_nlargest():
    series = pd.Series(np.arange(10))
    return series.nlargest(4)


print(series_nlargest())

Output:

> python series_nlargest.py
0    9
1    8
2    7
3    6
dtype: int64
